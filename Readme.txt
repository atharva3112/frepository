1. Before running the program make sure to download Python 2.7.4
2. After the installation, Sign in to WolframAlpha and create your own app id. Paste that app id in the variable app_id in the function OnEnter.
3. Download the wolfram Alpha API and Wikipedia API and let it complete the installation. 
4. After this, run the program and enjoy! 

NOTE: Your device should be connected to the internet for the chatbot to run properly.
